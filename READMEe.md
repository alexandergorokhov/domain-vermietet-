# Viermietet

Consumption counters.


### Prerequisites

```
Maven 3
Java 1.8
```
External libraries : codehaus.jackson, projectlombok, spring-boot. Full specification can be found in pom file.

### Installing

mvn clean install

Main class App.java.

### Running
You can run the application from console or IDE of your choise with:

mvn spring-boot:run
